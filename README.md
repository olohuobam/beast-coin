# $BEAST Coin — 3D Landing Page

## ⚡ Quick Start (3 commands)

Open your terminal / Git Bash, then:

```bash
# 1. Go into the project folder
cd beast-coin

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Then open your browser at: **http://localhost:5173**

---

## 📁 Project Structure

```
beast-coin/
├── index.html          ← HTML entry point
├── package.json        ← dependencies & scripts
├── vite.config.js      ← build config
└── src/
    ├── main.jsx        ← React root
    └── App.jsx         ← ENTIRE app (all sections + CSS)
```

---

## 🎨 What's Included

- **3D Interactive Coin** — Three.js coin that follows your mouse
- **Particle Background** — Animated network of dots
- **Glitch Text** — Cyberpunk-style animated title
- **Parallax Scrolling** — Depth on every section
- **Animated Counters** — Numbers count up on scroll
- **Tilt Cards** — 3D hover effect on all cards
- **Scroll Reveal** — Sections fade in as you scroll
- **Ticker Tape** — Scrolling announcement banner
- **Sections**: Hero → Ticker → Features → Tokenomics → Ecosystem → How to Buy → Community → Footer

---

## 🚀 Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder — ready to deploy to Netlify, Vercel, or any hosting.
